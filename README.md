# convaix

AI conversation exchange — store, search, share.

## Install

```bash
pip install convaix
pip install convaix[all]     # with embeddings + provider support
```

## Quick Start

```bash
convaix load ./conversations/    # Load conversation JSON files
convaix search "manifold"        # Hybrid search
convaix list                     # List loaded snapshots
convaix validate file.json       # Validate against schema v1.0
```
